Predict section


The Predict section is desined to provide tools for prediction of NMR data. It contains the following modules:

PREDICT module oreducts pseudo-contact shifts (PCSs), paramagnetic relaxation enhancements (PREs), and residual dipolar couplings (RDCs) induced in a protein by a paramagnetic moiety.

PATI module (currently under construction) provides tools for predicting residual dipolar couplings (RDCs) caused by molecular alignment in an anisotropic medium.

Diffusion Tensor Prediction/ELM module performs ab initio prediction of the rotational diffusion tensor of a protein directly from protein structure (atom coordinates) using the ellipsoid approximation of the solvent accessible surface of the protein.

More details are provided in the documentation for each module.
