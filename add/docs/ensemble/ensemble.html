Ensemble section


The Ensemble section contains modules for determining the conformational ensembles of proteins that match experimental data.

The underlying concept is as follows: the user inputs experimental data, dexp, and a matrix A containing predicted relevant data for each of the N conformers of a structural/conformational ensemble, and the software determines the relative populations/weights for each conformer (vector w) that provide the best match between the conformationally-weighted predicted data, dpred = A × w, and the experimental data. The agreement between dexp and dpred is quantified through χ2(w), where σ's represent experimental errors.

The relative weights for the conformers are determined by solving the optimization problem of minimizing dexp - dpred . The Ensemble section provides two distinct modules/nethods for achieving this: Sparse Ensemble Selection (SES) module and Maximum Entropy (MaxEnt) module.

The SES module determines the smallest number of conformers (from the initial input ensemble) necessary to match experimental data, i.e. it searches for vector w with the smallest number of non-zero entries that minimizes χ2(w).

In the MaxEnt module, dpred contains contributions from the entire input ensemble of conformers, and the minimized function contains, in addition to χ2(w), an entropy term as the regularizer to prevent overfitting. More details are provided in the documentation for each module.
