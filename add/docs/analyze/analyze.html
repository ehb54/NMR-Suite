Analyze section


The Analyze section is desined to provide tools for analysis of experimental NMR data. It contains the following modules:

ALTENS module provides analysis of residual dipolar couplings (RDCs) caused by anisotropy of protein orientations in solution. Given the experimental RDC data and protein atom coordinates, the module determines the alignment tensor of the protein.

PCS module provides analysis of pseudo-contact shifts (PCSs) induced in a protein by a paramagnetic moiety. Given the experimental PCS data and protein atom coordinates, the module determines the location of the paramagnetic moiety and characterizes its magnetic susceptibility tensor.

PRE module provides analysis of paramagnetic relaxation enhancements (PREs) caused by a paramagnetic moiety. Given the experimental PRE data and protein atom coordinates, the module determines the location of the paramagnetic center.

ROTDIF&Dynamics Analysis module provides analysis of nuclear spin relaxation data (T1, T2, heteronuclear NOEs etc) for various types of nuclei (15N, 13C) measured at a single or multiple magnetic fields. Given the experimental relaxation data and protein atom coordinates, the module determines the rotational diffusion tensor of the protein and characterizes local dynamics using model-free approach.

More details are provided in the documentation for each module.
